# Portfolio-Stocks-Performance-Analysis-Forecasting-with-CAPM
This project analyzes Portfolio stocks like—Reliance, TCS, HDFC Bank, Infosys, Bharti Airtel, and ITC—using five years of data. Applying CAPM, it calculates Beta to assess risk, compares actual vs. expected returns, and identifies undervalued or overvalued stocks. Results are visualized via Security Market Line and dashboards.
